Консольное приложение выполнение на языке C# которое ищет числа фибоначчи в заданном диапазоне.

## Особенности реализации

* **Поддержка сверхбольших чисел:** Использование типа `System.Numerics.BigInteger` позволяет вычислять значения, выходящие за пределы стандартных типов `long` и `ulong`.
* **Оптимизированный алгоритм:** Итеративный подход с временной сложностью $O(n)$ и минимальным потреблением памяти.
* **Валидация данных:** Обработка некорректного ввода (нечисловые значения, отрицательные диапазоны) через `TryParse`.

## Технологический стек

* **Язык:** C# (.NET Core 10.0)
* **IDE:** Visual Studio Code / Visual Studio
